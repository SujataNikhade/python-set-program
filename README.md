# python-set-program

This repository contains a collection of Python programs demonstrating operations on sets.
Each program is written in a simple and easy-to-understand way to help beginners learn core concepts.

## Topics Covered
### Create a Set

Program that takes user input and stores unique values inside a set.

### Iterate Over a Set

Loop through each element in a set.

### Add Member(s) to a Set

Add one or more elements to an existing set.

### Remove Item(s) From a Set

Remove one or multiple elements if present.

### Remove an Item Only if It Exists

Safe removal using membership check.

### Intersection of Two Sets

Find common elements between two collections.

### Union of Sets

Combine elements of both sets (without duplicates).

### Set Difference

Find elements present in one set but not the other.

### Symmetric Difference

Find elements that are NOT common between two sets.

### Check Subset

Check whether all elements of one set exist inside another.

### Remove All Elements

Clear a set by removing items one-by-one.

### Using Frozensets

Demonstration of immutable sets.

### Maximum and Minimum in a Set

Find smallest and largest values manually.

### Find Length of a Set

Count total elements without using len().

### Check if Value Exists in Set

Membership testing.

### Check Superset

Verify if one set contains all elements of another.

### Remove Intersection
